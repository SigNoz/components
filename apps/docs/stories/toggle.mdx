# Toggle

A two-state button that can be toggled on or off. For grouped toggles, use ToggleGroup with ToggleGroupItem or ToggleGroupSimple.

## How to use

### Standalone toggle

```tsx
import { Toggle } from '@signozhq/ui';
import { Bold } from '@signozhq/icons';

export default function MyComponent() {
  return (
    <Toggle aria-label="Toggle bold" defaultValue={false}>
      <Bold className="h-3 w-3" />
    </Toggle>
  );
}
```

### Controlled

```tsx
import { Toggle } from '@signozhq/ui';
import { Bold } from '@signozhq/icons';

export default function MyComponent() {
  const [bold, setBold] = React.useState(false);

  return (
    <Toggle aria-label="Toggle bold" value={bold} onChange={setBold}>
      <Bold className="h-3 w-3" />
    </Toggle>
  );
}
```
