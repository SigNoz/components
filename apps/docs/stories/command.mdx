# Command

Use `CommandDialog` to mount the command palette in a modal/portal, or compose the low-level primitives for inline palettes.

## How to use

### Inline Command

Use `Command`, `CommandInput`, `CommandList`, and `CommandItem` for an inline searchable list:

```tsx
import {
	Command,
	CommandInput,
	CommandList,
	CommandGroup,
	CommandItem,
	CommandEmpty,
} from '@signozhq/ui';

export default function MyComponent() {
	return (
		<Command>
			<CommandInput placeholder="Search…" />
			<CommandList>
				<CommandGroup heading="General">
					<CommandItem onSelect={() => {}}>Open settings</CommandItem>
					<CommandItem onSelect={() => {}}>Toggle sidebar</CommandItem>
				</CommandGroup>
				<CommandEmpty>No results.</CommandEmpty>
			</CommandList>
		</Command>
	);
}
```

## CommandDialog

Mount the command palette in a dialog for a global command menu (e.g. ⌘K):

```tsx
import {
	CommandDialog,
	CommandInput,
	CommandList,
	CommandGroup,
	CommandItem,
	CommandEmpty,
	CommandShortcut,
} from '@signozhq/ui';
import { useState } from 'react';

export default function MyComponent() {
	const [open, setOpen] = useState(false);

	return (
		<>
			<button onClick={() => setOpen(true)}>Open Command Dialog</button>
			<CommandDialog open={open} onOpenChange={setOpen} position="top" offset={110}>
				<CommandInput placeholder="Search or run a command…" />
				<CommandList>
					<CommandGroup heading="Quick actions">
						<CommandItem onSelect={() => setOpen(false)}>
							Open settings
							<CommandShortcut>⌘S</CommandShortcut>
						</CommandItem>
						<CommandItem onSelect={() => setOpen(false)}>Toggle sidebar</CommandItem>
					</CommandGroup>
					<CommandEmpty>No results.</CommandEmpty>
				</CommandList>
			</CommandDialog>
		</>
	);
}
```
