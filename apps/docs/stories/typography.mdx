# Typography

A flexible Typography component for rendering text with consistent sizing, weight, alignment, and truncation. Supports two variants: `title` (heading) and `text` (body).

## How to use

```tsx
import { Typography } from '@signozhq/ui';

export default function MyComponent() {
	return <Typography>Hello World</Typography>;
}
```

## Variants

### Title variant

Use `variant="title"` for headings. It renders as `<h2>` by default with bold weight. Combine with `as` to control the heading level.

```tsx
<Typography variant="title" as="h1" size="5xl">
  Main Heading
</Typography>
```

### Text variant

The default `variant="text"` renders as `<p>`. Use `as` to render as `<span>`, `<div>`, or `<label>`.

```tsx
<Typography size="lg">
  This is a paragraph of body text.
</Typography>
```

## Sizes

Available sizes: `xs` | `sm` | `base` | `lg` | `xl` | `2xl` | `3xl` | `4xl` | `5xl` | `6xl` | `7xl` | `8xl` | `9xl`

```tsx
<Typography size="xs">Extra small</Typography>
<Typography size="base">Base (default)</Typography>
<Typography size="4xl">Extra large</Typography>
```

## Weights

Available weights: `thin` | `extralight` | `light` | `normal` | `medium` | `semibold` | `bold` | `extrabold` | `black`

```tsx
<Typography weight="thin">Thin - 100</Typography>
<Typography weight="normal">Normal - 400</Typography>
<Typography weight="bold">Bold - 700</Typography>
```

## Truncation

Use `truncate` to clamp text after N lines with an ellipsis.

```tsx
<Typography truncate={2}>
  This long text will be clamped to 2 lines...
</Typography>
```

## Muted state

Use `muted` for secondary text like captions or helper text.

```tsx
<Typography muted>
  Helper text or captions
</Typography>
```

## Alignment

Control text alignment with the `align` prop.

```tsx
<Typography align="center" size="xl">
  Centered text
</Typography>
```
