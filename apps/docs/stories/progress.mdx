# Progress

The `Progress` component is used to display the current progress of an operation flow. It supports multiple sizes, custom colors, segmented steps, and status animations.

## How to use

```tsx
import { Progress } from '@signozhq/ui';

export default function MyComponent() {
	// A default continuous progress bar at 50%
	return <Progress percent={50} />;
}
```
