# Skeleton

Use `Skeleton` to show placeholder loading states while content is being fetched.
It supports animated pulse effects, configurable title and paragraph rows,
and sub-components for inputs, buttons, and avatars.

## Basic Usage

```tsx
import { Skeleton } from '@signozhq/ui';

export default function MyComponent() {
	return <Skeleton active title={{ width: '40%' }} paragraph={{ rows: 3 }} />;
}
```

## Sub-Components

```tsx

<Skeleton.Input active size="small" />
<Skeleton.Input active block />

// Button
<Skeleton.Button active size="small" />
<Skeleton.Button active block />

// Avatar
<Skeleton.Avatar active shape="circle" size={36} />

```
