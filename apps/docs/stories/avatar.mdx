# Avatar

The avatar component displays user profile images, initials, or icons with support for multiple sizes, shapes, and a loading skeleton state.

## How to use

```tsx
import { Avatar } from '@signozhq/ui';

export default function MyComponent() {
	return <Avatar size="lg" src="/user.png" alt="Jane Doe" />;
}
```
