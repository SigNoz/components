# Token Reference

Design tokens are the foundational building blocks of our design system—colors, spacing, typography, and more—expressed as data that can be shared across platforms and codebases.

This page is the **interactive token reference**: use the story list below to explore all tokens or focus on specific groups (Surface Levels, Actions, Callouts, Accents, and others). For full documentation on the token hierarchy, installation, surface levels, extending tokens, and migration from legacy tokens, see [Design System / Overview](?path=/docs/design-system-overview--docs).
